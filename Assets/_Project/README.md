# channel_play Unity Project

This folder contains all project-owned Unity assets.

## Structure

- `Art`: source and imported visual assets
- `Audio`: music, voice, and sound effects
- `Materials`: project materials and shaders
- `Prefabs`: reusable scene objects
- `Scenes`: Unity scenes
- `ScriptableObjects`: data-driven configuration assets
- `Scripts`: gameplay, networking, operator, UI, and logging code
- `Tests`: EditMode and PlayMode tests

## First Target

Create `School_MVP` with:

- one controllable player
- one operator camera
- clear team/name visibility
- OBS-recordable framing
