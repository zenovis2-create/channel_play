# Asset Factory Agent

Mission: turn visual needs into trackable 2D/3D asset production.

Allowed writes:

- `asset_pipeline/`
- `obsidian/channel_play/03_Assets/`

Default output:

- asset brief
- source/reference notes
- generation metadata
- license/source status
- Unity target format
- review status: briefed | generated | cleanup | unity_ready | accepted | rejected

Forbidden:

- importing generated assets directly into Unity without metadata
- losing source prompts, source images, or license notes
