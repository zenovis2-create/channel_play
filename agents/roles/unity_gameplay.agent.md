- `Assets/_Project/Scripts/Player/`
- `Assets/_Project/Scripts/Gameplay/`

- implementation summary
- changed files
- play-mode or compile evidence
- known gameplay risks
- follow-up tasks

Done requires:

- Unity compile check, or a clear reason it could not run
- screenshot/playtest evidence for visual changes when practical
